#  Exercício de Métodos em Java

Projeto simples desenvolvido em Java com o objetivo de praticar a criação e chamada de métodos (funções) no console.

##  Tecnologias utilizadas
- Java
- Apache NetBeans

##  Objetivo
Treinar a estrutura básica de um programa Java e o uso de métodos estáticos.

##  Funcionalidades
- Definição de métodos simples
- Chamada de métodos a partir do `main`
- Exibição de mensagens no console

##  Status
Projeto de estudo / exercício introdutório.
